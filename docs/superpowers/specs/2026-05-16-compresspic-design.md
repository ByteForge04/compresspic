# CompressPic - 在线图片压缩工具 设计文档

## 产品定位

免费在线图片压缩工具，浏览器本地处理，保护隐私。面向中文市场，极简工具风格。

## 目标市场

中文市场（国内用户），SEO 面向百度/搜狗，Vercel 海外部署。

## 技术方案

Next.js 14 (App Router) + TypeScript + Tailwind CSS

选择理由：生态成熟、Vercel 零配置部署、扩展性好（后续加工具页面只需新增路由）、SSG 模式 SEO 友好。

## 核心功能（MVP）

- 支持 JPG / PNG / WebP 格式输入
- 支持拖拽上传或点击选择文件
- 支持批量上传（最多 20 张）
- 质量滑块调节（1-100）
- 实时预览压缩前后对比
- 显示压缩率（如：从 2MB → 400KB，节省 80%）
- 一键下载压缩后的图片
- 批量下载（ZIP 打包）
- 隐私卖点：所有处理在浏览器本地完成，图片不会上传到任何服务器

## 技术架构

### 技术栈

- 框架：Next.js 14 (App Router)
- 语言：TypeScript
- 样式：Tailwind CSS
- 图片压缩：浏览器端 Canvas API + OffscreenCanvas（Web Worker 内）
- ZIP 打包：JSZip
- 部署：Vercel（静态导出 `output: 'export'`）

### 数据流

```
用户选择图片
  → FileReader 读取为 Blob
  → 创建 Image 对象获取尺寸
  → 传入 Web Worker
  → Worker 内用 OffscreenCanvas 重绘（按质量参数）
  → canvas.toBlob() 输出压缩后 Blob
  → 主线程展示预览 + 压缩信息
  → 用户点击下载 → URL.createObjectURL 下载
```

### 关键设计决策

- 压缩逻辑放在 Web Worker 中，避免大图片压缩时 UI 卡顿
- 不使用任何后端 API，纯静态导出，Vercel CDN 直接分发
- 所有页面 SSG 预渲染，SEO 友好
